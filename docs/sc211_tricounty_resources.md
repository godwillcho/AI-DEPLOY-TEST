## How to Use SC 211

- **Call:** Dial **2-1-1** from any phone (free, 24/7)
- **Text:** Send **HELP** to **211-211**
- **Online:** Visit [sc211.org](https://www.sc211.org)
- **Spanish:** Text **AYUDA** to **211-211**
